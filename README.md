# prayerscoring

## 概要

サッカー好きのために各々の選手や監督のベストショットの写真を共有し、いいねをもらい自己肯定感を高めるためのサービス

## 使い方

選手、監督、その他の写真を投稿したり、投稿された写真を見ることができます。

いいねすることや、選手だけ、監督だけと選択して見ることもできます。
